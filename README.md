# 💳 Simple ATM System

A simple web-based ATM simulation built with **HTML, CSS, and JavaScript**. This project demonstrates basic banking functionalities such as balance management, money transfers, transaction history, and PIN verification without requiring a backend or database.

## ✨ Features

- 💰 Display current account balance
- 🔄 Money transfer simulation
- 🔒 PIN confirmation before transfer
- 📜 Transaction history (Income & Expense)
- ➕ Add balance simulation
- 💾 Data persistence using LocalStorage
- 📱 Responsive and modern user interface
- 🔍 Search and filter transaction history
- 🎨 Mobile banking-inspired design

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **LocalStorage API**
- **Font Awesome**

## 📋 Available Features

### Dashboard
- Displays the current balance in Indonesian Rupiah (Rp).
- Updates balance in real-time.

### Transfer
- Enter recipient account number.
- Enter recipient name.
- Enter transfer amount.
- Confirm transaction using a 6-digit PIN.

### Transaction History
- View income and expense records.
- Displays:
  - Date & time
  - Transaction type
  - Description
  - Amount

### Add Balance
- Simulate incoming funds by adding balance manually.

## 💾 Local Storage

This project uses the browser's **LocalStorage** to save:

- Current balance
- Transaction history
- User data

Your data will remain available even after refreshing the page.
